Development and Version Control

Introduction

In this learning outcome, I explored front-end development using HTML, CSS, and JavaScript, and I integrated version control tools to enhance my workflow. My journey began with understanding the fundamentals of structuring, styling, and scripting web pages and evolved into effectively managing project versions using Git and GitHub. These tools allowed me to document changes, collaborate with my team, and maintain a clean development process.


Git and GitHub

After my first portfolio review, I realized the importance of proper version control. I started using Git for managing versions of my projects and GitHub as a platform for hosting my repositories. While my initial GitHub structure was disorganized, I have since restructured it to ensure that folders and files are clearly organized and easy to navigate. This helped me keep track of different iterations of my projects and collaborate effectively with teammates.
